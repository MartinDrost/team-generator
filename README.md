## Description

An API built to generate teams. The application is built using a Restful Nestjs server with sockets powered by Socket.io.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
